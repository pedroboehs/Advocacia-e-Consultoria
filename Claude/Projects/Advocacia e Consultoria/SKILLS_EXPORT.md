# Skills Export - Advocacia e Consultoria

Exportação completa de todas as skills customizadas do projeto em 2026-06-02.

## Skills Customizadas do Projeto

### 1. **humanizer-pt-br**
- **Descrição**: Remove traços de escrita gerada por IA de textos em português brasileiro, tornando-os mais naturais, humanos e com personalidade real.
- **Detecta e corrige**: Linguagem promocional, estruturas mecânicas, vocabulário genérico de chatbot, excesso de conectivos, frases de preenchimento, análises rasas com gerúndio
- **Ideal para**: Roteiros falados, artigos, posts, roteiros de vídeo, textos que precisam soar como escritos por seres humanos
- **Gatilhos**: humanizar texto, remover IA, texto artificial, soar natural, roteiro humano, reescrever texto, tirar traços de IA, texto robótico, parece IA

### 2. **squad-afiliados**
- **Descrição**: Gerencia toda a camada de receita - seleção de produtos afiliados por nicho, gestão de links, criação de info-produtos próprios e acompanhamento de comissões
- **Quando acionar**: Escolher produto afiliado para um post, recomendar plataforma de afiliados, criar info-produto simples, estruturar bio link page, analisar comissões por nicho
- **Gatilhos**: afiliado, Hotmart, Kiwify, comissão, monetização, produto, info-produto, link na bio, renda, venda, e-book, receita

### 3. **squad-analytics**
- **Descrição**: Monitora performance de cada conta e post. Identifica o que está viralizando, o que está convertendo e onde o funil está travando. Gera relatórios acionáveis
- **Quando acionar**: Revisar performance de posts, entender por que uma conta parou de crescer, identificar posts de alto desempenho, analisar funil de monetização, gerar relatório semanal
- **Gatilhos**: analytics, performance, métricas, crescimento, alcance, engajamento, conversão, relatório, funil, está funcionando, resultados, dados, análise

### 4. **squad-conteudo**
- **Descrição**: Cria conteúdo adaptado para o público brasileiro a partir de tendências internacionais. Produz roteiros de Reels, legendas otimizadas e define identidade de persona por conta
- **Quando acionar**: Precisar de roteiro de vídeo, legenda para post, calendário editorial, criar ou manter persona de uma conta, adaptar conteúdo do exterior para PT-BR
- **Gatilhos**: roteiro, legenda, conteúdo, Reel, TikTok, post, persona, calendário editorial, adaptar conteúdo, criar vídeo, script

### 5. **squad-copy**
- **Descrição**: Escreve todos os textos persuasivos do sistema - bio do Instagram, CTAs, copy de landing pages, sequências de venda e textos de afiliados. Especialista em conversão sem parecer publicidade
- **Quando acionar**: Escrever bio de perfil, criar CTA para post, montar página de vendas, redigir copy de info-produto, criar sequência de mensagens, refinar qualquer texto de venda
- **Gatilhos**: copy, bio, CTA, texto de venda, landing page, persuasão, converter, chamada, oferta

### 6. **squad-design**
- **Descrição**: Equipe de design para criação visual
- **Status**: Definir escopo específico

### 7. **squad-dev**
- **Descrição**: Equipe de desenvolvimento
- **Status**: Definir escopo específico

### 8. **squad-orquestrador**
- **Descrição**: Orquestra e coordena fluxos entre squads
- **Status**: Definir escopo específico

### 9. **squad-tendencias**
- **Descrição**: Identifica tendências e oportunidades
- **Status**: Definir escopo específico

### 10. **squad-trafego**
- **Descrição**: Gerencia estratégias de tráfego e crescimento
- **Status**: Definir escopo específico

### 11. **identidade-visual**
- **Descrição**: Guia de identidade visual do escritório de advocacia e consultoria
- **Acionar quando**: Mencionar criação de documentos visuais, logos, papelaria, apresentações, materiais de comunicação ou qualquer peça gráfica para o escritório
- **Gatilhos**: identidade visual, marca, branding, design, estilo visual, layout, papelaria executiva, cabeçalho de carta, materiais visuais
- **Referências**: Localização em `05. IDENTIDADE VISUAL/Referências/` (15 imagens)
- **Perfil estético**: Executivo e sofisticado, tons sóbrios (cinza, bege, marrom), detalhes refinados, design contemporâneo elegante

## Skills do Sistema Claude Code

Além das skills customizadas acima, o projeto tem acesso às seguintes skills nativas:

- **skill-creator**: Criar, modificar e melhorar skills; medir performance
- **update-config**: Configurar Claude Code via settings.json
- **keybindings-help**: Customizar atalhos de teclado
- **verify**: Verificar que mudanças funcionam
- **code-review**: Revisar código para bugs
- **simplify**: Simplificar código
- **fewer-permission-prompts**: Reduzir prompts de permissão
- **loop**: Executar prompt em intervalo recorrente
- **schedule**: Agendar agentes remotos
- **claude-api**: Construir apps com Claude API
- **run**: Executar o app do projeto
- **init**: Inicializar arquivo CLAUDE.md
- **review**: Revisar pull request
- **security-review**: Revisar segurança de mudanças
