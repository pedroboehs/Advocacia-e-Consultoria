## Perfil Estético

Com base nas referências visuais, o escritório segue este perfil:

### Atmosfera
- **Executivo e Sofisticado** — ambiente profissional de alto padrão
- **Discretamente Luxuoso** — detalhes refinados sem ostentação
- **Moderno com Elegância** — contemporâneo mas não minimalista

### Elementos Visuais
- **Iluminação**: Quente, aconchegante, bem iluminada
- **Mobiliário**: Peças de design, linhas limpas, materiais nobres (madeira escura, couro)
- **Cores Predominantes**: Tons sóbrios (cinza, bege, marrom), com detalhes em cores nobles
- **Estilo**: Design contemporâneo executivo, sem ser frio

### Arquétipos Capturados
- Ambientes de trabalho elegantes para profissionais sênior
- Interiores corporativos sofisticados
- Personagens de autoridade e inteligência
- Espaços que transmitem confiança, capacidade e tradição

## Como Usar Este Guia

Quando criar qualquer material visual para o escritório:

1. **Consulte as referências** — leia as imagens em `05. IDENTIDADE VISUAL/Referências/`
2. **Identifique padrões** — note paleta de cores, textura, estilo, atmosfera
3. **Aplique consistentemente** — ao desenhar, escrever layouts ou especificar cores, mantenha essa coerência visual
4. **Documente decisões** — se adicionar novas referências ou refinar o estilo, atualize este guia

## Exemplos de Aplicação

### Para Papelaria (Cabeçalho de Carta)
- Fundo: Branco ou creme suave
- Tipografia: Serif refinada (tradicional, autoridade)
- Cores: Cinza escuro ou marrom para texto; detalhes em ouro ou azul-marinho
- Espaçamento: Generoso, respirado, elegante
- Sem excesso de decoração

### Para Logos e Marca
- Formato geométrico e limpo
- Cores: 1-2 cores principais (acompanhar paleta)
- Proporcional e legível em qualquer tamanho
- Inspiração: Solidez, tradição, confiabilidade

### Para Apresentações
- Layout: Assimétrico elegante, não grade rígida
- Tipografia: Mix de serif (títulos) + sans-serif (corpo)
- Cores: Fundo claro, texto em tons sóbrios
- Imagens: Ambientes corporativos, interiores refinados
